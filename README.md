# Sajal Patil's Birthday Surprise Website 💖

Welcome to your partner's highly interactive and stunning birthday website! This project is crafted with a premium **"Midnight Dream"** design—a luxury dark mode with glowing glassmorphic elements, sparkling background stars, floating interactive hearts, a fun relationship quiz, a Polaroid sliding 3D card carousel, a typewriter letter reveal, and a virtual birthday cake where Sajal can blow out the candles to trigger a high-performance confetti explosion!

---

## 📂 Project Structure
- `index.html`: The semantic structure and pages of the surprise.
- `styles.css`: Visual styles, premium glassmorphism, responsive adjustments, and custom animations.
- `config.js`: **The control center** where you can edit the quiz, memories, and personal letter.
- `script.js`: The complex interactive magic (particles, slide controls, typing effect, candle extinguishing, and high-performance physics-based confetti).
- `assets/`: Folder to place your photos and optional song.

---

## 🛠️ Personalizing Your Website
You can customize everything in **`config.js`**! Open it in any text editor:

### 1. 📸 Adding Your Real Photos
By default, the website will display beautiful glowing placeholders with romantic icons. To add your real photos:
1. Find **5 of your favorite photos** together.
2. Crop them to a vertical shape if possible (portrait mode looks best inside Polaroids!).
3. Rename them to:
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
   - `photo4.jpg`
   - `photo5.jpg`
4. Place these files directly inside the `assets/` folder.
5. If your files have a different format (like PNG), open `config.js` and change `image: "assets/photo1.jpg"` to `image: "assets/photo1.png"`!

### 2. 🎵 Changing the Special Song
We have configured a beautiful acoustic piano track to stream automatically from a fast public CDN. If you want to use a specific song that means a lot to both of you:
1. Locate your song's MP3 file.
2. Rename it to `romantic_bg.mp3`.
3. Put it inside the `assets/` folder.
4. The website will automatically detect and play your custom song!

## 💻 How to View & Test Locally
To open and preview the website on your computer:
1. Simply double-click `index.html` to open it in your browser!
2. Alternately, you can run a local server if you want to test on your phone over Wi-Fi. (E.g. using VS Code "Live Server" extension, or running a python command like `python -m http.server` in this folder).

## 🚀 How to Host & Share with Sajal for FREE!
When you are ready to send this to Sajal, you can host it online completely free in just a few clicks:

### Option A: GitHub Pages (Highly Recommended & Super Easy)
1. Go to [GitHub](https://github.com/) and create a free account.
2. Create a new repository named `happy-birthday-sajal`.
3. Upload all the files from this folder (`index.html`, `styles.css`, `config.js`, `script.js`, and the `assets/` folder with your photos).
4. Go to **Settings** > **Pages** in your repository.
5. Under **Build and deployment**, select **Deploy from a branch**, set the branch to `main` (or `master`), and click **Save**.
6. GitHub will give you a private, beautiful URL (e.g., `https://aditipore.github.io/happy-birthday-sajal/`) to send to Sajal!

### Option B: Netlify (Drag & Drop)
1. Go to [Netlify](https://www.netlify.com/) and sign up for a free account.
2. Go to the dashboard and scroll to the bottom where it says "Want to deploy a new site without connecting to Git? Drag and drop your site folder here".
3. Drag the entire `Bithday` folder and drop it!
4. Netlify will deploy it in 5 seconds and give you a beautiful, shareable link! You can even change the domain name to something sweet for free!
